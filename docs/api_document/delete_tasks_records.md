# 删除记录

## 请求地址

- /review/v1/tasks/:task/records/:record

## 请求方式

- DELETE

## 地址参数

| 参数   | 是否必填 | 描述     |
| :----- | :------- | :------- |
| task   | 是       | `任务`ID |
| record | 是       | `记录`ID |

## 请求参数

| 参数 | 是否必填 | 描述                |
| :--- | :------- | :------------------ |
| ak   | 是       | `任务`的API访问秘钥 |
