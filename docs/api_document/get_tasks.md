# 读取任务信息

---

## 请求地址

- /review/v1/tasks/:task

## 请求方式

- GET

## 地址参数

| 参数 | 是否必填 | 描述     |
| :--- | :------- | :------- |
| task | 是       | `任务`ID |

## URL参数

| 参数 | 是否必填 | 描述                |
| :--- | :------- | :------------------ |
| ak   | 是       | `任务`的API访问秘钥 |

## 响应示例

``` json
{
	"num": 3,
	"records": ["jl0001", "jl0002", "jl0003"]
}
```

## 响应参数

| 参数   | 说明                     |
| :----- | :----------------------- |
| num    | 当前`任务`的`记录`数量   |
| record | 当前`任务`的`记录`ID数组 |
