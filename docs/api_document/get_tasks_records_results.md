# 读取记录查重结果

## 请求地址

- /review/v1/tasks/:task/records/:record/results

## 请求方式

- GET

## 地址参数

| 参数   | 是否必填 | 描述     |
| :----- | :------- | :------- |
| task   | 是       | `任务`ID |
| record | 是       | `记录`ID |

##  请求参数

| 参数 | 是否必填 | 描述                |
| :--- | :------- | :------------------ |
| ak   | 是       | `任务`的API访问秘钥 |

## 响应示例

``` json
{
	"ratio": "33",
	"sim_record": "jl0002"
}
```

## 响应参数

| 参数       | 说明         |
| :--------- | :----------- |
| ratio      | 相似比       |
| sim_record | 相似`记录`ID |
