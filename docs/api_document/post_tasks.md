# 创建任务

---

## 请求地址

- /review/v1/tasks

## 请求方式

- POST

##  URL参数

| 参数 | 是否必填 | 描述     |
| :--- | :------- | :------- |
| task | 是       | `任务`ID |

## 响应示例

``` json
{
	"ak": "3b2895b2-7e22-4692-940f-42de779a45fe"
}
```

## 响应参数

| 参数 | 描述                                                |
| :--- | :-------------------------------------------------- |
| ak   | `任务`及所属`记录`的API访问秘钥，后续操作都需要使用 |
